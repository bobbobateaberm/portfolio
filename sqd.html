SquareDodge

What is SquareDodge

It is my first game that I learned from youtube. It is a game that you are the player ( red square box ) . You will proceed to run through the level. During the journey , you wiil need to dodge the obstacle otherwise you will lose.

About

This game is made by Unity , a popular game engine for beginner-level developers. I watch a tutorial on youtube which have plenty of channels that made a series for beginner dev. The game use C# with Unity Libraries.

Download game file

Gameplay

A player will proceed through the level with obstacle. If you touched obstacles , you will be stopped and respawn at the begining of that level. If you fall of the level m you will be respawned to. There are 3 levels. Once you complete you are officially ended the game.
