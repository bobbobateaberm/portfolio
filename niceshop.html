NiceShop | Smoothie & Beverage

This WebApp is based from my own family business.We have been selling smoothies at the hospital canteen for over 10 years now. From my perspective , I see that many nurses and medical staff were needed to take some walk to order our smoothies from another compound. So I come with the idea that we could make an automation system that can take orders online and effectively handle loads of request from customers.

Try our Websites Here!

Client-Side Features

User Separation

Separate users session by generating random IDs for each.

Cart Systems

Add and edit items into the cart.There is a calculated price sums up.

Delivery Systems

Delivery to user's location and Pick-up option at onsite.

Checkout Systems

Sending order details of each items and its orderIDs to Admin Panel.

OrderIDs Generation

IDs for order confirmation for each checkout. This is randomly generated.

Friendly Interface

Sorting items and search bar provided. Tags for each items.

Responsive Designs

Works on smaller screen and devices. Some element are cut out in phones.

Advertisment Section

Advertisement slider at the top of the page. Autoslides are implemented.

Adminstration Features

Inventory System

Adding , edting items in the inventory and real time updates.

Uploading items

Form consist of name , price , quantity , tags , images.

Order Handling

Receiving orders from various user. Details are kept in the table.

Friendly Interface

All data are kept in sortable table. Each section comes with tag sorting.

Responsiveness

We have mobile webapp consist of shop sites and admin panel supported for more device .both comes with adjusted size to work on mobile devices. The site is responsive on most device and can be adjust the element accordingly.
