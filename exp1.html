Chula MOOC 2 Python | IoT | Flask | DialogFlow

This course provided me with decent knowledge on using Python and its libraries. The project is about an AI robot gardener which will respond to moisture and light inputs. The mechanism involves using GUI sliders to input light and moisture values. These values are then sent to a Python file to respond with different conditions. The response is then sent to Line, which is connected to Dialogflow and the server. For example, if you set the value of light to 60 in the GUI, then if you open the Line chat and ask if the sensor light bulb is on, the Line bot will respond with "off" depending on what value the GUI is set to.

KMUTT DDAY Wifi in Organization

I've learned the basics of how Wi-Fi works and have an opportunity to configure a real network switch! The class is teaching from the basics of Wi-Fi origin, types, and ultimately to understand the mechanism. The basics, like Wi-Fi being a frequency signal, there will be different channels for each type. In the case of 2.4 GHz, the recommended channels are 1, 11, 6 since they do not interfere with other frequencies, which would affect the speeds. We learned to set up a basic network for an isolated environment, working on switches and access points. Also, configuring the network with Aruba Tool.

KMUTT DDAY AI Object Recognition

The AI class. The professors teach us about different types of how AI detect objects. Whether it is detecting gestures, images, and objects. We mainly focus on object recognition. It will include different techniques to detect objects in images. The clearly main point of the course is classification and segmentation techniques. The segmentation is even cooler on classifying and separating objects that also in the same class. For example, there are two cats in the picture, instance segmentation will accurately define that there are 2 different cats in the picture. We use YOLO model for study case. It is also capable to display the possibility of that object to be that exact object.
